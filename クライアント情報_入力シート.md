## 2. EmailJS（フォーム送信機能）

フォームからの問い合わせメールを受信するために必要です。

**取得先:** https://www.emailjs.com/（無料プラン: 月200通まで）

| 項目 | ご記入欄 | 取得場所 |
|------|----------|----------|
| Public Key | | Account > API Keys > Public Key |
| Service ID | | Email Services > 対象サービスのID |
| Template ID | | Email Templates > 対象テンプレートのID |

> **テンプレート作成時に使える変数:**
> `{{from_name}}` 送信者名 / `{{from_email}}` メール / `{{from_phone}}` 電話 / `{{message}}` 本文 / `{{quote_summary}}` 見積内容
